Template Engine

I expanded upon the custom template engine we bagan using for the main page. Every page of the site now uses templating in some way, including the pages from lab 1! All pages will now have the same base style, requiring only writing the inner html for each new page.

def apply_template(title: str, content: str, css = [], scripts = []): with open('templates/template.html', 'r') as template_file: template = template_file.read() stylesheet_html = '' stylesheet_template = '<link rel="stylesheet" href="$url" />' for stylesheet in css: stylesheet_html += stylesheet_template.replace('$url', stylesheet) script_html = '' script_template = '<script src="$url"></script>' for script in scripts: script_html += script_template.replace('$url', script) template = template.replace('!pageTitle!', title) template = template.replace('!stylesheets!', stylesheet_html) template = template.replace('!main!', content) template = template.replace('!scripts!', script_html) return template

The python code that enables this takes in 4 parameters:

  1. A title for the page
  2. The HTML content for the page
  3. A list of links to CSS sources for the page (can be internal or external)
  4. A list of links to JavaScript sources for the page (can be internal or external)
